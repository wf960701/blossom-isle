## 🧱 技术栈

- **框架**: Next.js 16 + TypeScript (App Router)
- **样式**: Tailwind CSS + 自定义 CSS
- **图标**: Font Awesome 6 Free（`@fortawesome/fontawesome-free`）
- **字体**: Noto Sans SC + Noto Serif SC + Playfair Display
- **后端**: Supabase (PostgreSQL + Auth + Storage)
- **部署**: Vercel

## 🎨 设计主题

- **基调**: 温暖粉色系
- **品牌色**: `rose-400` `pink-500` 渐变
- **背景**: 暖杏色 `#fef6f3`，带微妙的粉色径向渐变
- **点缀**: 花瓣飘落动画、毛玻璃导航、圆角卡片
- **氛围**: 温柔、知性、像花园一样舒适

## 🗄️ 数据库（Supabase）

项目 ID: `artwyllxkkoabmgzupdn`

```sql
articles       → 文章 (id, title, slug, content, excerpt, cover, status, published_at)
tags           → 标签 (id, name, slug)
article_tags   → 文章-标签关联 (article_id, tag_id)
friends        → 友链 (id, name, url, description, avatar, sort_order)
site_config    → 站点配置 (id, key, value)
```

## 📄 页面路由

| 路由 | 页面 | 类型 |
|------|------|:----:|
| `/` | 首页（文章列表 + 标签云 + 英雄区） | 服务端 |
| `/articles/[slug]` | 文章详情 | 服务端 |
| `/tags` | 所有标签 | 服务端 |
| `/tags/[tag]` | 按标签筛选文章 | 服务端 |
| `/friends` | 友链展示 | 服务端 |
| `/search` | 搜索页面 | 客户端 |
| `/feed.xml` | RSS 订阅 | 路由 |
| `/login` | 管理登录 | 客户端 |
| `/admin` | 后台管理首页 | 服务端 |
| `/admin/articles/new` | 写文章 | 客户端 |
| `/admin/articles/[id]/edit` | 编辑文章 | 混合 |
| `/admin/tags` | 标签管理 | 混合 |
| `/admin/friends` | 友链管理 | 混合 |
| `/admin/settings` | 站点设置 | 混合 |
